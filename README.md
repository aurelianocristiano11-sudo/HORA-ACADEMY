# HORA ACADEMY - Landing Page OCPCA 2026

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Faça upload de todos os ficheiros desta pasta na raiz do repositório.
3. Confirme que o ficheiro principal se chama `index.html`.
4. Vá em Settings > Pages.
5. Em Source, escolha:
   - Deploy from a branch
   - Branch: main
   - Folder: /root
6. Clique em Save.

Depois, o GitHub vai gerar o link público da landing page.
